1. В начальном состоянии страница состоит из одного белого блока заполняющим 100% страницы. 2. Каждый блок имеет свой порядковый номер (не отображается). 3. При клике по произвольному блоку, соответствующий блок должен разделиться на два блока в точке клика вертикальным или горизонтальным разрезом, таким образом, чтобы соотношения площадей (абсолютное, отображаемое) новых блоков отличалось минимально. 4. Больший блок сохраняет свой номер, меньший блок получает новый. 5. Цвет нового наименьшего блока должен быть случайным, но существенно отличаться от большего блока и отличаться от всех цветов других блоков. 6. После отрисовки новых блоков, скрипт должен отправить (через ajax) на произвольный сервер информацию о блоках (порядковый номер, цвет, абсолютный размер в пикселах) в формате json 7. При ресайзе окна, все блоки меняют размер пропорционально. 8. Текст тестового задания отображается поверх блоков в абсолютном центре страницы, полностью при любом размере и на кликабельность по блокам не влияет.
